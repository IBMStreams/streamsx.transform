Operator Modify

IBMStreams com.ibm.streamsx.transform Toolkit > com.ibm.streamsx.transform 1.2.0 > com.ibm.streamsx.transform > Modify

This operator is similar to Functor, but modifies the input tuple instead of creating a new one. Like Functor, it has an optionally filter. If your input type and output type are the same, using this operator instead of Functor.

The Modify operator can be an operator within the reachability graph of a consistent region. It cannot be the start of a consistent region.
