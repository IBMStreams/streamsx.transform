Streamsx.transform

Toolkit that contains general-purpose operators for processing tuples.

View on GitHub Download .zip Download .tar.gz

Toolkit description

This toolkit contains general purpose operators for processing tuples. Variations of Join, Aggregate, and Functor fit in this toolkit's scope. Operators that work on the streams and don't inspect the tuples themselves go in streamsx.plumbing. Currently, there is only one operator, Modify.
